# Sensor-HC-SR04-With-Oled
HC-SR04 adalah sensor ultrasonik jarak yang bekerja dengan cara memancarkan gelombang suara berfrekuensi tinggi (40 kHz) dan mengukur waktu pantulannya (echo) saat gelombang mengenai objek.  Dari waktu pantulan itu, sensor menghitung jarak antara sensor dan objek menggunakan kecepatan suara.
